Unity LOL VS
Ein persönliches Lern‑ und Experimentierprojekt in Unity, das grundlegende Mechaniken eines MOBA‑ähnlichen Spiels nach dem Vorbild von League of Legends nachbildet. Dieses Repository dient ausschließlich als privates Read‑Only‑Projekt, um meinen eigenen Fortschritt zu dokumentieren und meinen Code zentral zu versionieren.

🎯 Zweck des Projekts
Dieses Projekt ist nicht zur Nutzung durch andere gedacht.
Es dient ausschließlich:
- meinem eigenen Lernfortschritt
- dem Ausprobieren von Gameplay‑Mechaniken
- dem Strukturieren und Verbessern meines C#‑Codes
- dem sicheren Speichern meiner Scripts über GitHub
Es ist kein vollständiges Spiel, kein Framework und keine Vorlage für andere Projekte.

🧩 Inhalt des Repositories
Das Repository enthält nur die C#‑Scripts, die ich im Rahmen des Projekts entwickle.
Unity‑Assets, Prefabs, Scenes oder Projektdateien sind bewusst nicht enthalten.
Aktuelle Kernskripte:
- Enemy_Movment.cs — Gegnerbewegung und KI‑Grundlogik
- follow.cs — Verfolgungslogik
- in_range.cs — Reichweitenprüfung
- Move_to_opponent.cs — Bewegung auf Gegner zu
- Movment.cs — Basisbewegung
- Projectil.cs — Projektilverhalten
- Spawner.cs — Minion‑Spawner
- tower.cs — Tower‑Angriffslogik

🛠️ Technologien
- Unity (Version nach Bedarf)
- C#
- Visual Studio 2022
- Git / GitHub (nur Scripts werden versioniert)

🔒 Nutzung & Lizenz
Dieses Repository ist nicht zur Weiterverwendung, Modifikation oder Veröffentlichung durch Dritte vorgesehen.
Es dient ausschließlich als persönliches Lernprojekt und als private Dokumentation.

📌 Status
Das Projekt befindet sich in laufender Entwicklung und wird nach Bedarf erweitert.
Es gibt keinen Anspruch auf Vollständigkeit, Stabilität oder Nutzbarkeit.
